## 📚 Structure Overview

This repository is organized into **6 main levels** that guide you from basic C++ concepts through Object-Oriented Programming:

1. **[1_Fundamentals/](./1_Fundamentals/)** - Variables, I/O, Control Flow
2. **[2_Loops_and_Iterations/](./2_Loops_and_Iterations/)** - While, For, Nested Loops
3. **[3_Arrays_and_Data_Structures/](./3_Arrays_and_Data_Structures/)** - Array Operations & DSA Basics
4. **[4_Functions/](./4_Functions/)** - Modular Programming & Recursion
5. **[5_Object_Oriented_Programming/](./5_Object_Oriented_Programming/)** - Classes, Encapsulation, Problem Solving
6. **[6_Advanced_Concepts/](./6_Advanced_Concepts/)** - Templates, Pointers, STL

🎓 Topics Covered
### Fundamentals
- Data types and variables
- Input/output with `cin`/`cout`
- Arithmetic operations
- Conditional statements (`if`/`else`, `switch`)

### Loops
- `while` loops (pre-condition)
- `for` loops (counted iteration)
- `do-while` loops
- Nested loops

### Arrays & DSA
- Array declaration and initialization
- Array indexing and bounds
- Searching (linear search, maximum/minimum)
- Common operations (sum, reverse, frequency)

### Functions
- Function prototypes and definitions
- Parameters and return values
- Default arguments
- Recursion (factorial example)

### Object-Oriented Programming
- Class definition and instantiation
- Constructors
- Encapsulation (`private`/`public` members)
- Getters and setters
- Real-world problem modeling

### Advanced Concepts *(Coming Soon)*
- Pointers and memory management
- Function templates
- Standard Template Library (STL)

## 🎯 Learning Path

### For Beginners
Start with **1_Fundamentals** → **2_Loops_and_Iterations** → **3_Arrays** → **4_Functions**

### For OOP
Progress to **5_Object_Oriented_Programming** with its sub-sections:
- Classes and Objects
- Encapsulation (Private/Public)
- Problem Solving with Classes

### For Advanced Topics
Explore **6_Advanced_Concepts** once comfortable with OOP.
