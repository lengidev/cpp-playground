# C++ Playground 🚀

This repository documents my journey learning and practicing C++.  
It contains small programs, experiments, and examples created while exploring both core and modern C++ concepts.

Originally started as a simple learning dump, this repo is now being cleaned up and organized as I improve my understanding of C++ and Git.

### What You'll Find Here
- Core C++ fundamentals
- Object-oriented programming concepts
- Standard Template Library (STL)
- Memory management and pointers
- Modern C++ features (C++17 / C++20)
- Small experiments and practice problems

### Purpose
- Practice and reinforce C++ concepts
- Experiment freely without worrying about perfection
- Track progress over time
- Learn proper project organization and version control

### Building & Running
Most examples can be compiled using a standard C++ compiler
