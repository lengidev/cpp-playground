# 2. Loops and Iterations

Master different loop constructs to automate repetitive tasks:
- **While Loops**: Pre-condition loops for unknown iteration counts
- **For Loops**: Definite iteration with initialization, condition, increment
- **Nested Loops**: Loops within loops for complex patterns

## Learning Path
1. Understand `while_loops/` - when you don't know how many times you'll iterate
2. Move to `for_loops/` - when you know the exact number of iterations
3. Explore nested loops for matrix-like operations

## Key Concepts
- Loop structure and flow
- Break and continue statements
- Nested iterations
- Loop efficiency and patterns
