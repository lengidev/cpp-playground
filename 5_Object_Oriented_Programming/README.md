# 5. Object-Oriented Programming

Design programs using objects, classes, and encapsulation:
- **Classes & Objects**: Blueprint-based object design
- **Encapsulation**: Private data and public interfaces
- **Problem Solving**: Real-world problem modeling with OOP

## Learning Path
1. Understand `classes_and_objects/` - how to define and use classes
2. Learn `encapsulation/` - hiding data and controlling access
3. Solve `problem_solving/` - apply OOP to real scenarios
4. Progress to `advanced/` - inheritance, polymorphism, virtual functions

## Key Concepts
- Classes and objects
- Constructors and destructors
- Encapsulation (private/public/protected)
- Inheritance and polymorphism
- Virtual functions and abstract classes
