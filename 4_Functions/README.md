# 4. Functions

Break down complex problems into manageable, reusable pieces:
- **Basics**: Function declaration, definition, calls, and parameters
- **Advanced**: Default parameters, function overloading, recursion
- **Templates**: Generic function programming (C++17+)

## Learning Path
1. Learn `basics/` - how to write and call functions
2. Explore `advanced/` - specialized function techniques
3. Study templates for generic, reusable code

## Key Concepts
- Function prototypes and definitions
- Parameters and return values
- Scope and lifetime
- Recursion and base cases
