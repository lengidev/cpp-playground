# 1. Fundamentals

This section covers the basic building blocks of C++:
- **Variables & I/O**: Understanding data types, input/output operations
- **Control Flow**: If/else statements, switch cases, conditional logic
- **Input/Output**: Using `cin` and `cout` effectively

## Learning Path
1. Start with `variables_and_io/` to understand basic syntax
2. Move to `control_flow/` for decision-making logic
3. Practice writing programs that take user input and respond conditionally

## Key Concepts
- Data types (int, double, string, bool)
- Variable declaration and initialization
- Standard input/output (cin, cout)
- Conditional statements (if, else if, else, switch)
